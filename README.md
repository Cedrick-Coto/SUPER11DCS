# SUPER11DCS — Supermarket Stock Management System

A desktop application built in C# for managing supermarket inventory, including warehouse stock, brands, and product categories.

## Features

- 📦 Warehouse stock management
- 🏷️ Brand registration and tracking
- 🗂️ Product category organization
- 📊 Inventory administration panel

## Tech Stack

- **Language:** C#
- **Platform:** .NET (Windows Desktop)

## About

Academic project developed as part of a computer science course. Focuses on inventory management logic and data organization for a retail/supermarket context.
